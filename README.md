### 学习Java各种框架或编程思想时的一些Demo,涵盖市面上各种Java主流技术
1.  **Spring** - 轻量级的**企业级应用开发容器**和**一站式框架**，核心特性为IoC（控制反转）与AOP（面向切面编程）。
2.  **SpringBoot** - **快速构建和简化Spring应用**的框架，采用“约定优于配置”的理念，简化了项目搭建与部署。
3.  **Servlet** - **Java Web开发的底层基础组件**，用于处理客户端请求并生成动态响应，是Java EE规范的核心。
4.  **SpringMVC (SSM整合)** - 基于Spring的**Web层MVC框架**，用于解耦Web层组件；SSM即Spring+SpringMVC+Mybatis的经典组合。
5.  **Mybatis** - **半自动化的持久层框架**，通过简单的XML或注解将Java对象与SQL语句进行映射，避免几乎所有的JDBC代码。
6.  **MybatisPlus** - **Mybatis的增强工具**，在Mybatis基础上只做增强不做改变，提供通用的CRUD、条件构造器等，简化开发。
7.  **Maven** - **项目管理和构建自动化工具**，基于POM（项目对象模型），用于依赖管理、项目构建、文档生成等。
8.  **FastDFS** - **轻量级的分布式文件系统**，主要用于文件存储、访问及负载均衡，特别适合以文件为载体的在线服务（如图片、视频）。
9.  **RabbitMQ** - **基于AMQP（高级消息队列协议）的开源消息代理**，用于在分布式系统中存储和转发消息，实现应用解耦、异步处理和流量削峰。
10. **Nio** - **Java的非阻塞I/O（New I/O）**，提供了与标准I/O不同的高速、面向缓冲区的I/O操作方式，是实现高性能网络通信的基础。
11. **Netty** - **异步的、事件驱动的网络应用框架**，用于快速开发可维护的高性能协议服务器和客户端（如RPC、HTTP服务器）。
12. **Dubbo** - **高性能的Java RPC（远程过程调用）框架**，致力于提供透明化的远程方法调用、智能负载均衡、服务自动注册与发现。
13. **Zookeeper** - **分布式协调服务**，为分布式应用提供一致性服务，如配置维护、命名服务、分布式同步和组服务。
14. **Shiro** - **强大且易用的Java安全框架**，提供认证、授权、加密和会话管理功能，以保护应用程序的安全。
15. **DockerFile** - **自动化构建Docker镜像的脚本文件**，包含了一系列指令，用于定义应用程序的环境和启动方式。
16. **MinIo** - **高性能、兼容S3协议的对象存储服务器**，适用于存储图片、视频、日志文件、备份数据等非结构化数据。
17. **Redis** - **基于内存的高性能键值对存储数据库**，支持多种数据结构，广泛用于缓存、消息队列、分布式锁等场景。
18. **SpringData JPA** - **Spring基于ORM框架（如Hibernate）、JPA规范的封装**，用于简化数据访问层的实现，减少持久层代码量。
19. **SpringSession** - **提供跨多个应用实例管理用户会话信息的解决方案**，解决了分布式或集群环境下Session共享的问题。
20. **多线程和JUC** - **Java并发编程的核心**，JUC（java.util.concurrent）包提供了大量用于处理多线程的工具类，如线程池、锁、原子类等。
21. **常见设计模式** - **软件设计中常见问题的典型解决方案**，如单例、工厂、代理、策略等模式，是对代码经验的总结与抽象。
22. **SSO** - **单点登录解决方案**，指在多个应用系统中，用户只需登录一次即可访问所有相互信任的应用系统。
23. **SpringCloud+SpringCloudAlibaba** - **微服务架构的一站式解决方案**，提供了服务发现、配置中心、网关、熔断器等组件，用于构建分布式系统。
24. **Swagger** - **遵循OpenAPI规范的接口文档自动生成+功能测试工具**，允许开发人员描述API结构，并生成交互式文档。
25. **Quartz分布式定时任务** - **功能丰富的开源作业调度库**，可以集成在几乎任何Java应用中，用于创建简单或复杂的定时任务。
26. **EasyExcel** - **快速、简洁的处理Excel文件的Java工具**，旨在解决大文件内存溢出问题，提供低内存占用的读写操作。
27. **Elasticsearch** - **基于Lucene的分布式、RESTful风格的搜索和数据分析引擎**，常用于全文检索、结构化搜索和分析。
28. **Thrift** - **跨语言RPC服务调用框架**，由Facebook开发，通过IDL（接口定义语言）定义服务，支持多种编程语言间的通信。
29. **SpringSecurity** - **功能强大且高度可定制的身份验证和访问控制框架**，专注于为Java应用提供认证和授权服务，是Spring项目的安全标准。
30. **java新特性** - **指Java语言各版本（如8、11、17、21等）引入的新功能**，包括Lambda表达式、Stream API、模块化、记录类型等。
31. **Gradle** - **基于Apache Ant和Apache Maven的自动化构建工具**，使用Groovy或Kotlin DSL编写构建脚本，提供了更灵活的性能优化。
32. **MapStruct** - **Java注解处理器**，用于生成类型安全的、高性能且无依赖的Bean映射（POJO之间转换）代码，在编译时处理映射注解。
